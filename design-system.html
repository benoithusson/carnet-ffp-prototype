Components

Reusable UI components extracted from existing screens. All components follow BEM naming convention.

Text Input

Standard text input field with label, placeholder, and focus states.

HTML

<div class="[block]__input-wrapper">
  <label for="input" class="[block]__label">Label</label>
  <input
    id="input"
    type="text"
    class="[block]__input"
    placeholder="Placeholder par défault"
  />
</div>

CSS (BEM Classes)

.[block]__input-wrapper { display: flex; flex-direction: column; gap: var(--space-xs); }
.[block]__label { font-size: var(--font-size-sm); font-weight: var(--font-weight-medium); }
.[block]__input {
  height: 48px;
  padding: var(--space-m) var(--space-l);
  border: var(--border-width-sm) solid var(--border-input);
  border-radius: var(--radius-lg);
}
.[block]__input:focus {
  border-width: var(--border-width-xl);
  border-color: var(--content-primary);
}
