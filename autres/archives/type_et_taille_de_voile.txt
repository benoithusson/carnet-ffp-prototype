### **Type de voile**
https://www.figma.com/design/tgIdYqaDPkIU24TXabUcvo/Design?node-id=2506-5340&t=xCPQYci3qJN3mMKx-1
- Information: 
    - The sections by canopy type are sorted alphabetically, just like in WhatsApp.
    - When I scroll down, the search bar disappears. It reappears when I scroll up.
    - When I select an option, it is saved in memory unless it is deleted.
    - When I select an option and click Validate, it is displayed on the previous page, Authorized Canopy Size and Type, under the corresponding label with the following styles:
        - Color: #5A5F65
        - Spacing: 8px
        - Font size: 14px
    - Once I selected an option, I can validate.

- Information for the container with the letters: 
    - On scroll, for each container containing the alphabet letters used to categorize the sections, when it reaches the header, it stays sticky just below it with a transparent background — just like in WhatsApp.
    - The letter container then disappears when the next letter’s container reaches it: when the next letter’s container hits the bottom border of the previous one, the previous container is pushed upward under the header and disappears, while the new letter container takes its place.
    - The slight box shadow on the bottom border is therefore hidden by the new letter container. 
    - Use this link as an example: https://www.figma.com/design/tgIdYqaDPkIU24TXabUcvo/Design?node-id=2520-21365&t=xCPQYci3qJN3mMKx-1

- Information for the container with the selected options:
    - When I select options, chips are displayed in the header below the search bar.
    - The headers are displayed one after another. The list is horizontally scrollable.
    - Spacing between all header elements: 8px.
    - When I click the chip’s “Close icon”: it removes the tag and deselects the option in the list. The checkbox returns to an unchecked state.
    - When I uncheck an option in the list, the corresponding chip is removed from the list.
    - Use this link as an example: https://www.figma.com/design/tgIdYqaDPkIU24TXabUcvo/Design?node-id=2520-21749&t=xCPQYci3qJN3mMKx-1
